# atxhomerescue.com

Production static site for ATX Home Rescue, served from this repo via GitHub Pages on the `main` branch.

## Preview locally

```bash
python -m http.server 8080
# then open http://localhost:8080
```

## Deploy

Push to `main`. GitHub Pages publishes automatically.
